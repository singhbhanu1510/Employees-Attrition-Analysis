Employee Attrition Analysis Power BI project

Data Quality Checks

* Checked for missing values and duplicates.
* Verified employee IDs for uniqueness.
* Standardized department, job role, and education categories.
* Validated age, salary, and experience ranges.
* Created calculated columns and measures for attrition rate.

### Business Problems Analyzed

* Which departments have the highest attrition?
* Which age groups are leaving most frequently?
* Does salary impact employee turnover?
* Which job roles have the highest resignation rates?
* Are there gender-based attrition differences?

### Key Insights

* Sales and HR departments showed higher attrition.
* Employees aged 26–35 had the highest turnover.
* Lower salary bands experienced greater attrition.
* Certain job roles had significantly higher resignation rates.
* Attrition was more common among employees with shorter tenure.

### Project Description

Developed an end-to-end Employee Attrition Analysis dashboard in Power BI by performing data cleaning, transformation, modeling, and visualization. Identified key drivers of employee turnover through interactive dashboards, helping HR teams improve retention strategies and workforce planning.
